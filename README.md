# AoGate - Generic Gatekeeper

AoGate 是一个通用的 TCP 服务安全外壳（代理认证网关）。它通过密钥对（支持 Ed25519, RSA, ECDSA）进行身份验证。
只有持有正确私钥并通过 Launcher 验证的客户端，其 IP 地址才会被允许访问后端目标服务器。
你可以将其作为不修改游戏服务器的情况下对 Minecraft/Terraria 公网服务器进行保护的一种选择。
或是使用ipv6替代原服务器不支持ipv4端口的套壳网关使用。

## 使用示例  
1. 在 `25564` 端口上运行Minecraft服务器，使用 `.\server -l 25565 -b 127.0.0.1:25565` 进行端口保护。
2. 使用 `.\keygen John` 为John创建密钥对，默认保存在 `keys` 目录下。
3. 使用 `.\packager` 交互式打包一个launcher(启动器)给John，在其中的交互式步骤中填入John的密钥序号、您的服务器地址等。
4. 将生成的 `launcher_John_<Platform>.zip` 发送给您的好友John，John在运行`.\launcher`后即可与您部署的AoGate网关进行认证，在您设置的认证过期倒计时结束前，他可直接加入您的服务器进行游玩。

## 优势
 - 支持众多主流桌面平台和架构。
 - 无需修改游戏服务器或后端程序即可运行。
 - 占用内存极低(<10MB)，对设备负担轻。
 - 管理密钥便捷，直接对文件系统进行操作。

## 功能特性

*   **安全防护**：保护 TCP 服务（如游戏服务器、数据库等），防止未经授权的访问。
*   **多密钥支持**：支持 Ed25519 (默认), RSA (2048), ECDSA (P-256) 算法。
*   **组件分离**：包含 Server, Launcher, Keygen, Packager 四个独立组件，各司其职。
*   **配置灵活**：支持命令行参数覆盖配置文件，支持自动生成默认配置。
*   **便捷分发**：内置打包工具，可一键生成包含客户端和私钥的 ZIP 包。
